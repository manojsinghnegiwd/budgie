# PWA Icons

This directory should contain the following icon files for the PWA:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

You can generate these icons using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Or any image editor that can export PNG files

The icons should have a budgie bird theme and work well on both light and dark backgrounds.
